# Naming Convention

The purpose of this document is to provide a guide for naming the components of the application's
architecture. The naming decisions have followed some of the rules of Hexagonal Architecture, and
they are mandatory.

The conventions are ordered from the highest level of abstraction to the lowest.

| Abstraction level | Responsibility                                                     | Prefix / suffix           | Example/s                                                                                                                   |
|:------------------|:-------------------------------------------------------------------|:--------------------------|:----------------------------------------------------------------------------------------------------------------------------|
| Package           | Application                                                        |                           | `com.entity.project.application`                                                                                            |
|                   | Configuration                                                      |                           | `com.entity.project.infrastructure.configuration`                                                                           |
|                   | Domain                                                             |                           | `com.entity.project.domain`                                                                                                 |
|                   | Data Access                                                        |                           | `com.entity.project.infrastructure.persistence`                                                                             |
|                   | Exception                                                          |                           | `com.entity.project.domain.exception`                                                                                      |
|                   | Presentation                                                       |                           | `com.entity.project.infrastructure.api`                                                                                     |
|                   | Security                                                           |                           | `com.entity.project.infrastructure.security`                                                                                |
|                   | Utility                                                            |                           | `com.entity.project.utils`                                                                                                  |
| Interface         | Behaviour abstraction                                              | -able                     | `Auditable`, `Comparable`, `Iterable`                                                                                       |
|                   | DB access abstraction                                              | -Repository               | `ResourceRepository`                                                                                                        |
|                   | External API client abstraction                                    | -Client                   | `HolidayCalendarApiClient`, `WeatherApiClient`                                                                              |
|                   | Mapping                                                            | -Mapper                   | `ResourceMapper`                                                                                                            |
| Concrete class    | API                                                                | -Controller               | `ResourceController`                                                                                                        |
|                   | Business logic orchestration                                       | -Service                  | `ResourceService`                                                                                                           |
|                   | Configuration                                                      | -Config                   | `ResourceConfig`                                                                                                            |
|                   | Configuration properties                                           | -Properties               | `ResourceProperties`                                                                                                        |
|                   | Enumerated type                                                    | -Type                     | `ResourceType`                                                                                                              |
|                   | Event                                                              | -(ActionInPastTense)Event | `ResourceCreatedEvent`                                                                                                      |
|                   | Exception                                                          | -Exception                | `ResourceException`                                                                                                         |
|                   | JPA Entity                                                         | -Entity                   | `ResourceEntity`                                                                                                            |
|                   | Test (integration)                                                 | -IntegrationTest          | `ResourceIntegrationTest`                                                                                                   |
|                   | Test (unit)                                                        | -Test                     | `ResourceMapperTest`, `ResourceServiceTest`                                                                                 |
|                   | Utility                                                            | -Utils                    | `ResourceUtils`                                                                                                             |
|                   | Validation                                                         | -Validator                | `ResourceValidator`                                                                                                         |
| Record            | Creation                                                           | -Creation                 | `ResourceCreation`                                                                                                          |
|                   | Deletion                                                           | -Deletion                 | `ResourceDeletion`                                                                                                          |
|                   | Full update                                                        | -Update                   | `ResourceUpdate`                                                                                                            |
|                   | Non-CRUD operation input                                           | -Request                  | `ResourceCancellationRequest`                                                                                               |
|                   | Non-CRUD operation output                                          | -Response                 | `ResourceSearchResponse`                                                                                                    |
|                   | Partial update                                                     | -Patch                    | `ResourcePatch`                                                                                                             |
|                   | Search                                                             | -Search                   | `ResourceSearch`                                                                                                            |
